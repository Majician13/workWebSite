T-Check Error Codes

  • sh sys – Show System – Shows PCTs, Status, Time/Date, and FITs
  • sh site – Show Site – Shows site
  • Login OPWTECH (or TECH) – Username to login
  • 1PV2opwfms – password to use with Login OPWTECH (or TECH)
  • install pct 1 pos 1 – installs pct 1 pos 1 if locked out (as shown by parenthesis around number when using sh sys)
  • remove pct 1 – Uninstalls pct 1 from FSC. This may be required if card reader continuously requests pump #. Removing and then reinstalling pct is a common resolution.
  • install pct 1 – Installs/re-installs pct 1
  • sh trans where date = 091222 (and time > 1600) – used to print to screen all transactions for the date 09/12/22 (optional time is greater than, to show only trans after a certain time on specified date).
  • set time – used to set FSC to proper time.
  • set date – used to set FSC to proper date.
